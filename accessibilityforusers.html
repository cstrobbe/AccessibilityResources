This page contains reference to tips and advice for computer users. Most computer users are no technical experts and are unaware of what they can do to improve the accessibility of the technology they use.

Accessibility Tips for Desktop and Laptop Computers

  • Microsoft Accessibility:
    • 10 Tips for the Awkward Age of Computing: tips for personalising and customising your Windows PC.
    • Guide for Individuals with Vision Impairments.
  • Cat Ellis: How to use the accessibility tools in Windows 10, TechRadar, May 2017. Overview of accessibility features, for end users.
  • Bill Holton (AFB): Using a Computer with a Visual Impairment: A Beginner's Guide to Computer Accessibility.
  • Lasa Information Systems Team : Mac OS X Accessibility Features (October 2006).
  • Jon Gooday (AbilityNet): Accessibility on Linux desktops (May 2007).
    Note: for more up-to-date information about Linux, see the page Linux Accessibility Resources.
  • Configurer Windows pour les malvoyants, 01.04.2014.
  • Windows 10 Build 16215 brings significant accessibility improvements including colour filters for the colourblind, MSPowerUser, 09.07.2017.
  • Becky Gibson: Keyboard Navigation in Mac Browsers, Weba11y, 07.07.2014.

Browser Features and Extensions for End Users

  • Faulkner, Steve: Short note on getting spaced out with SC 1.4.12 Text Spacing, The Paciello Group Blog, 24.05.2018. Contains a bookmarklet that can be used to apply text spacing properties to webpages.
  • BBC: My web my way - Making the web easier to use (archived content; this site is no longer being updated).
  • W3C: Better Web Browsing: Tips for Customizing Your Computer (last updated in May 2010).
  • W3C: Contacting Organizations about Inaccessible Websites (last updated in May 2010).
  • Mozilla Support: Accessibility features in Firefox - Make Firefox and web content work for all users.
  • Media Access Australia: How to turn on accessibility features in Google Chrome.
  • Media Access Australia: How to turn on accessibility features in Opera.
  • chrome extensions for disabilities masterpost (June 2015): a list of extensions for people with reading disabilities.
  • UI Options Plus (UIO+) is an extension for Chrome developed by the Fluid Project. It allows people to customize websites by tweaking text size, line spacing, contrast and other features.
  • Landmarks: browser extension for Firefox, Chroma and Opera. See also the article by Graham Armfield: Browser Extensions can Deliver Landmark Navigation for All Keyboard Users (08.05.2018).

Dyslexia

Fonts for Dyslexia

The following fonts were specially developed for readers with dyslexia.

  • OpenDyxlexic (free).
  • Dyslexie font (not free, except for home use).
  • Lexie Readable (formerly Lexia Readable; not free, except for personal, educational or charity use).
