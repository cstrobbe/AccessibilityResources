General Resources

  • Funka Nu: Mobile accessibility guidelines.

  • IBM AbilityLab Mobile Accessibility Checker (see also: Techworld: "IBM launches accessibility test tool for mobile apps", 4 March 2015).
  • W3C:

    • Mobile Accessibility: How WCAG 2.0 and Other W3C/WAI Guidelines Apply to Mobile (work in progress) by the Mobile Accessibility Task Force (Mobile A11Y TF).

Java Mobile Accessibility

  • Peter Korn: Java Mobile Speaks! Report on our LWUIT accessibility work for Java ME, Peter Korn's Weblog (Oracle), 13 October 2011.
    • The javax.accessibility package exists in the AEGIS LWUIT version (no download available at the moment); it seems that this was not integrated in the official version of LWUIT.
  • Peter Korn: Java Spotlight on mobile accessibility with LWUIT, Peter Korn's Weblog (Oracle), 29 November 2011.
  • LWUIT on Java.net (no changes since 2012?). LWUIT is still available for download (see Oracle Java for Mobile and Embedded Devices - Downloads, but it is not clear what Oracle will do with Java ME in the future.)
