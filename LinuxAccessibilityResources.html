Specific Linux Distributions and Desktop Environments

  • Sonar GNU/Linux: a Linux distribution that focuses on assistive technologies.
  • Vinux: a Linux distribution based on Ubuntu that is optimised for the needs of blind and partially sighted users.
  • MATE Desktop Environment. MATE is a fork of the now unmaintained GNOME 2. When the GNOME Project released GNOME 3, much of the accessibility work had to be redone. For this reason, many people with disabilities continued using GNOME 2 and, more recently, MATE. Some consider MATE the best choice for users with serious visual impairments (including blindness). (However, GNOME 3 has a better magnifier and a black-on-white contrast theme.) Linux Mint includes MATE as one of its default desktop environments.
